# Netflix Content Analysis Dashboard

Analysed 8,000+ Netflix titles across genres, ratings, countries and release 
trends using Power BI, Tableau and Excel to uncover content growth patterns 
and regional distribution insights.

## Tools Used
- Power BI
- Tableau
- Microsoft Excel

## Key Insights
- Identified top genres and content growth trends by year
- Mapped regional content distribution across countries
- Analysed ratings breakdown across Movies vs TV Shows
- Uncovered peak content addition periods on Netflix
